# Happy Birthday Web App 🎂✨

A beautiful, interactive, and responsive romantic birthday countdown website crafted with modern HTML, CSS, and vanilla JavaScript. Designed specifically to celebrate a very special person's birthday.

---

## 🌟 Features

* **Interactive Custom Cursor**: A smooth custom dual-ring pointer that interacts with buttons and inputs.
* **Dynamic Visual Effects**: Soft falling rose petals and floating heart emojis in the background.
* **Live Countdown Timer**: Calculates remaining days, hours, minutes, and seconds based on user input. Automatic triggers when the special day arrives.
* **Custom Message Update**: Allows you to customize the birthday greeting message, sender, and recipient names in real-time.
* **Interactive Confetti Burst**: Fun and celebratory confetti animation triggered by actions and target goals.
* **Scroll Reveal Animation**: Smooth fading and sliding transitions as you scroll through memories and wishes.
